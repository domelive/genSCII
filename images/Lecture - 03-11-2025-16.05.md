#### Variabili Di Default
+ Sono molto simili ai parametri statici del C++.
+ Normalmente fanno riferimento ad oggetti immutabili.
+ Vedere cosa succede quando si modifica un parametro di default dalle slide.
	+ Sostanzialmente sono delle variabili statiche.
		+ Se vengono modificate si tiene l'ultimo valore di esse e si continua con quello a rifare le operazioni.
#### Moduli
+ Ogni modulo √® un file.
+ Il nome del file mi permette di identificare il modulo.
	+ Il nome senza l'estensione viene memorizzato nella variabile `__name__`.
		+ Viene cambiato in `__main__` se il modulo viene invocato direttamente.
		+ `if __name__ == '__main__': ...`.
+ Per accedere al modulo devo mettere l'istruzione `import module`.
	+ Posso cambiare il path di dove prendo i moduli, basta che esistano anche nell'altro luogo.
+ Per caricare i nomi nel namespace posso usare la sintassi `from module import name`.
	+ Posso usare anche `*` per importare tutti i nomi tranne quelli che iniziano con `__`.
		+ Non vuol dire che non posso accedervi, ma devo prima esplicitarlo per poterlo fare.
	+ Posso anche fare degli alias con `as` per cambiare un nome all'interno di un namespace.
#### Namespace
+ Definisce tutti i nomi che sono associati con un modulo.
+ Posso dare visibilit√† ad essi con la chiamata alla funzione `dir(module)`.
	+ Elenca tutti i nomi del namespace passato.
